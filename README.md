## 💡 Proposta de Valor

O ReCiclo é um aplicativo que conecta pessoas que desejam doar itens com aquelas que precisam deles, promovendo sustentabilidade e reaproveitamento de recursos.

O principal benefício para o usuário é a facilidade de encontrar ou doar objetos de forma rápida, organizada e acessível, incentivando práticas sustentáveis e ajudando a reduzir o desperdício.

## 📱 Descrição das Telas

**1. Tela de Feed (Home)**
Apresenta uma lista de itens disponíveis para doação. O usuário pode visualizar rapidamente os objetos, com imagem e informações básicas.

**2. Tela de Filtros de Busca**
Permite refinar a busca por itens com base em critérios como categoria, distância e estado de conservação, facilitando encontrar o que realmente precisa.

**3. Tela de Detalhes do Item**
Mostra informações completas sobre o item selecionado, como descrição, imagens e dados do doador. Também possibilita iniciar interesse pelo item.

**4. Tela de Cadastro de Novo Item**
Permite que o usuário cadastre um objeto para doação, adicionando fotos, descrição, categoria e outras informações relevantes.

**5. Tela de Chat**
Espaço onde doador e interessado podem conversar para combinar detalhes da entrega ou retirada do item.

**6. Tela de Perfil / Conquistas**
Exibe informações do usuário e suas conquistas dentro do app, como selos relacionados à participação e ações sustentáveis.

## 🔄 Jornada Principal do Usuário

1. O usuário acessa o aplicativo e visualiza o feed de itens disponíveis.
2. Ele pode utilizar os filtros para encontrar algo específico.
3. Ao encontrar um item de interesse, acessa a tela de detalhes.
4. Demonstra interesse e inicia uma conversa pelo chat com o doador.
5. Após combinar os detalhes, realiza a retirada do item.
6. Ao concluir a ação, pode receber conquistas (selos) dentro do aplicativo.
